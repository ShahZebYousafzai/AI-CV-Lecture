# AI & Computer Vision — Complete Training Curriculum

A structured one-on-one training program covering Computer Vision, Deep Learning, LLMs, Generative AI, and AI Agents. Sessions are held **Monday / Wednesday / Friday**, each lasting **1–1.5 hours** with a ~50/50 split between theory and hands-on coding.

---

## Program Overview

| Stat | Value |
|---|---|
| Total Phases | 10 |
| Total Duration | ~28 Weeks |
| Total Sessions | 83 |
| Total Hours | ~100 Hours |
| Milestone Projects | 10 |
| Final Capstone | 1 |

**Tools & Stack**

| Category | Tools |
|---|---|
| Language | Python 3.10+ |
| CV Libraries | OpenCV, Ultralytics YOLOv8, MediaPipe, PaddleOCR, InsightFace |
| DL Framework | PyTorch + torchvision |
| GenAI Stack | HuggingFace Diffusers, Transformers, PEFT, bitsandbytes |
| LLM / Agent Stack | LangChain, LangGraph, FAISS, ChromaDB, Streamlit |
| Deployment | FastAPI, Docker, ONNX, TensorRT, RunPod |
